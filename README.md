# EML Viewer Web App

Web app to view the contents of an .eml file as JSON. All processing is done locally on your machine. No data leaves your device.

## Requirements

Install wasm-pack

## Instructions

To run locally:

```
git clone https://github.com/tebuevd/eml-viewer
cd eml-viewer
bun install
bun run build-wasm
bun run dev
```
